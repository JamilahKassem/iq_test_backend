# IQ Test Backend

This repository contains the backend server for the IQ Test application. It is built using **Node.js** and **Express**, featuring real-time communication via **WebSockets** and data persistence with **MySQL**.

---

## 🛠 Tech Stack

* **Node.js**: JavaScript runtime environment.
* **Express**: Fast, unopinionated web framework for API routing.
* **ws**: Simple and efficient WebSocket implementation for real-time data.
* **mysql2**: MySQL client for Node.js with support for prepared statements.

---

## 📋 Prerequisites

Before you begin, ensure you have the following installed on your local machine:

1.  **Node.js** (LTS version recommended): [Download here](https://nodejs.org/)
2.  **MySQL Server**: Ensure you have a database instance running locally or hosted remotely.
3.  **npm**: Usually comes bundled with Node.js.
